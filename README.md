# 国学少年

> 一个基于VUE开发的微信H5页面，包含报名、签到、上传图片、微信支付、生成分享海报及后台管理统计功能。

## Build Setup

```bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```

## 应用到的技术或框架
1. vux，基于Vue的UI框架；
2. qrcode.js，生成二维码图片；
3. html2canvas，dom转图片，用于生成海报。
